# ODPS SDK for Java 介绍

MaxCompute(ODPS) SDK for Java 允许开发者将他们的Java应用程序与阿里云的 MaxCompute 服务集成，MaxCompute
是一个大规模并行处理的数据仓库系统。

该SDK提供了一组丰富的API，使您能够轻松地与 MaxCompute 进行交互，执行各种数据处理任务。
